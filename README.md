# ecommerce-previsao-vendas-powerbi
Análise e previsão de resultados de um e-commerce utilizando Power BI, com foco em métricas de negócio como receita, ticket médio, lucratividade, retenção de clientes e desempenho de categorias de produtos para apoiar decisões estratégicas de marketing, estoque e precificação a partir de um painel interativo para o time de negócios.
